Gridea Linux 客户端小记

2024-05-10

如您所见，本站的静态生成基于Gridea。

笔者不久前将自己的环境从Window迁移到Ubuntu Linux中。本文将记述在Ubuntu Linux环境下Gridea的部署……

部署与设置
获取软件包
前往GitHub获取最新软件包。

https://github.com/getgridea/gridea/releases
截止到发稿日，最新版本为0.9.3。

不幸的是，v0.9.3并为提供传统的deb安装包，仅释出了适用于snap包管理的snap软件包。

下载snap软件包到~/Download文件夹。

安装软件
将工作目录跳转至Download文件夹

使用Snap进行安装。由于官方未提供包含签名的.assert文件，所以需要使用选项--dangerous。

cd ~/Download
sudo snap install ./gridea_0.9.3_amd64.snap --dangerous
等待安装就绪。

在安装完成后，Gridea图标将会在菜单中显示。

添加主题
将主题文件夹复制到~/snap/gridea/current/Documents/Gridea/themes文件夹下，重新启动Gridea即可。

/snap/gridea/current/Documents/Gridea/themes是/snap/gridea/x1/Documents/Gridea/themes的一个链接。
